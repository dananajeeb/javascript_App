# نظام الكشف الذكي عن رسائل التصيد الاحتيالي (Phishing Detection System)

نظام متكامل مخصص للاختبار الفعلي للكشف عن محاولات التصيد الاحتيالي في البريد الإلكتروني عبر تحليل النصوص ومطابقتها ديناميكياً.

## 📌 عن المشروع

صُمم هذا الاختبار البرمجي ليحاكي آلية الكشف الذكي عبر مراحل متسلسلة؛ حيث يقوم النظام بقراءة وتحليل نص رسالة البريد الإلكتروني برمجياً، ثم إجراء مطابقة ديناميكية للنصوص مع الجداول المعرفية المحددة لكل مرحلة من مراحل الهجوم.

يتولى الكود البرمجي فحص المحتوى بدقة لاكتشاف وجود "الكلمات المفتاحية المحظورة" وسياقاتها الأمنية، ومن ثم تصنيف الرسالة برمجياً وتحديد هويتها إما كـ:
* **رسالة شرعية (Legitimate)**
* **محاولة تصيد احتيالي (Phishing)**

---

## 🛠️ التقنيات المستخدمة

* **لغة البرمجة:** JavaScript
* **الواجهة:** واجهة تطبيق مخصصة للاختبار الفعلي (الشكل رقم 5.15)

---

## 🚀 آلية العمل (Workflow)

1. **إدخال النص:** قراءة نص رسالة البريد الإلكتروني المراد فحصها.
2. **التحليل البرمجي:** مطابقة النصوص ديناميكياً مع الجداول المعرفية الخاصة بكل مرحلة من مراحل الهجوم.
3. **فحص المحتوى:** التفتيش عن الكلمات المفتاحية المحظورة والسياقات الأمنية المرتبطة بها.
4. **التصنيف النهائي:** إظهار النتيجة فوراً (Legitimate أو Phishing).
